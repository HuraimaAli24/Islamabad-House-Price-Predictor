This repository contains a complete end-to-end machine learning project that predicts residential property prices in Islamabad, Pakistan. The project covers every stage of a real ML pipeline: data collection through web scraping, data cleaning and feature engineering, exploratory data analysis, multi-model training and comparison, and deployment as an interactive web application using Gradio.
Web scraping — Property listings are scraped from Zameen.com using Selenium (headless Chrome) and BeautifulSoup. The scraper navigates through paginated results for Islamabad, scrolls each page to trigger lazy-loaded listings, and extracts price, area, location, number of bedrooms and bathrooms, listing title, and date added. A random delay between pages prevents rate limiting. The scraper targets 400 listings across up to 20 pages.
Data cleaning and feature engineering — Raw price strings (e.g. "PKR 10.5 Crore", "PKR 85 Lakh") are converted to numeric PKR values. Area strings (Marla, Kanal, Sq. Ft., Sq. Yd.) are standardised to square feet. Bedrooms and bathrooms are converted to numeric, missing values filled with medians, and unrealistic values removed. Top and bottom 1% price outliers are dropped. Location is cleaned and Label Encoded. Property type is inferred from the listing title and also encoded.
EDA — Price distribution, area distribution, bedroom count, and property type breakdown are visualised in a 2×2 subplot. A price vs area scatter plot reveals general trends.
Model training — Six regression models are trained and compared: Linear Regression, Decision Tree, Random Forest, Gradient Boosting, XGBoost, and CatBoost. All are evaluated on MAE, RMSE, and R² score. The best performing model is automatically selected for the app.
Gradio web app — A polished multi-step interface (welcome → property details form → price estimate) lets users enter area, location, bedrooms, bathrooms, and property type and receive an instant price estimate displayed in Crore or Lakh format with a property summary card.
Tech stack: Python, Selenium, BeautifulSoup, scikit-learn, XGBoost, CatBoost, Gradio, pandas, matplotlib — running in Google Colab
